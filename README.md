# 政企大客户数字化运营工具

通用的政企客户数字化运营管理系统，提升政企客户管理效率至70%以上。

## 功能特性

### 基础功能
- ✅ 客户信息标准化录入（批量导入/导出 + 字段校验）
- ✅ 按收入/潜力的客户智能分层 + 跟进策略匹配
- ✅ 多维度数据可视化看板（客户分层、跟进进度、商机漏斗）
- ✅ 跟进台账导出 + 节点提醒

### 自动化运营
- 📧 多渠道消息推送：支持短信、邮件、APP推送
- 📅 活动运营管理：创建政企活动、报名统计、线索收集
- 🔔 跟进提醒：自动提醒客户跟进时间和节点

### 政企场景深化
- 🏷️ 需求标签库：政企需求标签（校园网建设/信息化改造/上云服务），自动关联对应解决方案模板
- 📊 竞品分析模块：竞品信息录入（对手名称/方案漏洞/价格），辅助制定差异化方案

### 权限与安全
- 👥 角色权限控制：管理员/销售/运营三种角色
- 📝 操作日志：完整记录所有操作行为
- 💧 数据导出水印：导出的数据带有导出人信息

### 效率优化
- 📋 模板复用：预设「政务/园区/国央企」三类客户录入模板，减少重复填写
- ⚡ 快捷操作："一键标记高潜客户""快速生成跟进话术"按钮

## 技术栈

- **状态管理**: Zustand

## 智能分层算法

系统基于多维度智能评估客户价值：

1. **收入维度** (0-30分)
   - 年营收5000万以上：30分
   - 年营收2000-5000万：25分
   - 年营收1000-2000万：20分

2. **规模维度** (0-25分)
   - 大型：25分
   - 中型：15分
   - 小型：5分

3. **需求维度** (0-25分)
   - 根据需求数量评分

4. **紧急度维度** (0-20分)
   - 根据客户状态和跟进日期评分

**分层规则**：
- A级（高潜客户）：总分≥70分
- B级（重点客户）：总分≥40分
- C级（普通客户）：总分<40分
